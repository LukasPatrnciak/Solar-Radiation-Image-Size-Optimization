# Optimization of Input Sky Image Size for Solar Radiation Estimation Using CNN
This project focuses on designing and evaluating custom Convolutional Neural Network (CNN) architectures for solar radiation estimation based on sky images. The main objective is to determine the optimal input image dimensions that minimize prediction error while considering model complexity, training time, and hyperparameter configurations.

Different CNN structures with varying levels of complexity are implemented and trained using sky photographs as input data. The performance of each model is evaluated across multiple image resolutions. In addition to prediction accuracy, training duration and the number of epochs required for convergence are also analyzed.

Hyperparameters such as learning rate, batch size, optimizer type, and network depth are systematically tuned to achieve the best possible performance. The project provides a comparative analysis of model error metrics in relation to input image size and computational cost.

Inputs
 - Sky images (photographs of the sky)

Outputs
- Optimal input image dimensions for each selected CNN architecture
- Prediction error metrics (e.g., MAE, RMSE)
- Training time and number of epochs
- Tuned hyperparameter configurations

Objectives
- Design and implement custom CNN architectures
- Optimize input image size for solar radiation estimation
- Tune training hyperparameters
- Evaluate model performance and training efficiency
- Analyze the relationship between image resolution and prediction accuracy
